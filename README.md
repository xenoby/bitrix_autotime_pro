# bitrix_autotime_pro
Это расширение превращает рутинное управление рабочим временем в полностью автономный процесс
Оно разработано специально под обновленный интерфейс Битрикс24 и учитывает все особенности работы в «облаке».

Основные функции:
Умное расписание: Автоматический старт дня, уход на обед, возобновление работы и завершение смены.
Учёт перерывов: Поддержка полноценного цикла «Работа — Обед — Работа — Выход» с настройкой времени каждой фазы.
Календарь дней недели: Выбор конкретных дней для срабатывания (скрипт не беспокоит вас в выходные).
Живые уведомления: Всплывающие сообщения о новых чатах с отображением имени отправителя, текста сообщения и его аватарки.
Контроль отчетов: Уведомление о необходимости заполнения еженедельных отчетов (Weekly Reports).
Интеллектуальный кликер: Уникальная система поиска кнопок, которая распознаёт статус дня по тексту на странице («Работаю», «День завершён»), исключая лишние клики.

Преимущества:
Двухпоточный мониторинг: Раздельные функции для контроля времени и мониторинга чатов.
Быстрый доступ: Клик по уведомлению мгновенно переключает браузер на вкладку с Битриксом.
Прозрачный лог: В меню расширения всегда виден статус последнего успешного действия.
Адаптивность: Настроено под актуальную разметку Bitrix Air (включая работу с Shadow DOM и фреймами).
